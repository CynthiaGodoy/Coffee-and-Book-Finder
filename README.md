# Book-Club-Finder

## Project Description

An interactive calendar app for organizing book club meetups and information with a map to show where the meetup is.

## Deployment | Main Objective

The main objective of this project is to learn how to effectively collaborate with other developers on a web application that is hosted on GitHub. To avoid confict multiple branches are created within a single project. 

## What is Used
    * index.html
    * script.js
    * style.css with use of [Bulma](https://bulma.io/)
    * (x2) Third-Pary API's 
        -[Open Library] (https://www.programmableweb.com/api/open-library-books-rest-api)
        -[Open Street Map](https://wiki.openstreetmap.org/wiki/API_v0.6)

## How to Use the Project

✅ The main page of the App with allow a user to find a book club by Location. In the demo we only have one location. More can be added later.

✅ When Denver is clicked, the Denver Book Club page will be opened. 

✅ As a book club member you will be able to view the specific book club calendar.

✅ As a book club member you will be able to click on an event on the calander and be able to view the map/location of the event.

✅ As a book club member you will be able to add books from the Open Library to the calendar.

✅ As a book club member you will be able to add the event and specific book to the calendar.
